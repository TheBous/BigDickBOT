## Installation

- You need Node >= 12.20
- run 
```bash
yarn
```

## Using ENV

- Create your own .env and fill it with these values: 
```bash
API_SECRET=
API_PASSPHRASE=
API_KEY=
BOT_TOKEN=
```

## Start

Start it with 
```bash
yarn start
```
